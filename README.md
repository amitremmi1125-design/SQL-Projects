# SQL Database Projects

This repository contains multiple SQL mini-projects designed to demonstrate database design, table creation, relationships, and query execution using MySQL.

## Projects Included

1. **Traffic Violation Management System**
   - Manage vehicle violations, fines, and payments.
   - Tables: Users, Vehicle Owners, Vehicles, Violations, Payments.

2. **Online Shopping System**
   - Manage customers, products, orders, and payments.
   - Tables: Customers, Categories, Products, Orders, Order Details, Payments.

3. **Student Management System**
   - Manage student information, courses, and marks.

4. **Event Management System**
   - Handle event registration, participants, and ticket payments.

## Technologies Used
- MySQL
- SQL (DDL, DML)
- MySQL Workbench

## Features
- Database creation
- Table relationships using Primary and Foreign Keys
- Insert sample data
- SQL queries using JOIN, GROUP BY, and Aggregate Functions

## Purpose
These projects are created for learning SQL, practicing database design, and building portfolio projects.
